2.3 Opérateurs et expressions

Dans cette leçon, nous allons explorer les opérateurs en Python et comment les utiliser pour créer des expressions. Les opérateurs sont des symboles spéciaux qui effectuent des opérations sur des variables et des valeurs.

Opérateurs arithmétiques

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques de base:


# Addition
a = 5 + 3  # a vaut 8

# Soustraction
b = 10 - 4  # b vaut 6

# Multiplication
c = 4 * 5  # c vaut 20

# Division
d = 20 / 4  # d vaut 5.0 (notez qu'il s'agit d'un float)

# Division entière
e = 20 // 3  # e vaut 6 (la partie décimale est tronquée)

# Modulo (reste de la division)
f = 20 % 3  # f vaut 2

# Puissance
g = 2 ** 3  # g vaut 8 (2 puissance 3)

Opérateurs de comparaison

Les opérateurs de comparaison sont utilisés pour comparer deux valeurs:


# Égal à
a == b  # Renvoie True si a est égal à b

# Différent de
a != b  # Renvoie True si a est différent de b

# Supérieur à
a > b  # Renvoie True si a est supérieur à b

# Inférieur à
a < b  # Renvoie True si a est inférieur à b

# Supérieur ou égal à
a >= b  # Renvoie True si a est supérieur ou égal à b

# Inférieur ou égal à
a <= b  # Renvoie True si a est inférieur ou égal à b

Opérateurs logiques

Les opérateurs logiques sont utilisés pour combiner des expressions conditionnelles:


# ET logique
a and b  # Renvoie True si a et b sont True

# OU logique
a or b  # Renvoie True si a ou b est True

# NON logique
not a  # Renvoie True si a est False, et False si a est True

Opérateurs d'affectation

Les opérateurs d'affectation sont utilisés pour assigner des valeurs à des variables:


# Affectation simple
a = 5

# Affectation avec addition
a += 3  # Équivalent à a = a + 3

# Affectation avec soustraction
a -= 2  # Équivalent à a = a - 2

# Affectation avec multiplication
a *= 4  # Équivalent à a = a * 4

# Affectation avec division
a /= 2  # Équivalent à a = a / 2

Exercice pratique

Essayez de résoudre l'exercice suivant pour tester votre compréhension:

Exercice: Calculez la moyenne de trois nombres et vérifiez si elle est supérieure à 10.

Commencez avec les valeurs a = 12, b = 8, et c = 14.

  • Calculez la moyenne
  • Vérifiez si elle est supérieure à 10
  • Affichez un message approprié

a = 12
b = 8
c = 14

# Calcul de la moyenne
moyenne = (a + b + c) / 3

# Vérification
est_superieur_a_10 = moyenne > 10

# Affichage du résultat
if est_superieur_a_10:
    print(f"La moyenne est {moyenne}, elle est supérieure à 10.")
else:
    print(f"La moyenne est {moyenne}, elle n'est pas supérieure à 10.")

# Résultat:
# La moyenne est 11.333333333333334, elle est supérieure à 10.

Lucas Martin il y a 2 jours

Est-ce que quelqu'un peut m'expliquer la différence entre la division normale (/) et la division entière (//) en Python ?

Bernard Dupont Enseignant il y a 1 jour

Bonjour Lucas, la division normale (/) renvoie toujours un nombre à virgule flottante (float), même si le résultat est un nombre entier. Par exemple, 4 / 2 donnera 2.0.

La division entière (//) tronque la partie décimale et renvoie un entier. Par exemple, 5 // 2 donnera 2 (et non 2.5).
